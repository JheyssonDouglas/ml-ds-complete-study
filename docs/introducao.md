# 📘 Introdução à Ciência de Dados

## 📌 O que é Ciência de Dados?

Ciência de Dados é uma área multidisciplinar que combina **estatística**, **programação**, **análise de dados** e **conhecimento de negócio** para **extrair valor e gerar conhecimento a partir de dados**.

Imagine um monte de dados brutos — como cliques em um site, registros de vendas, ou postagens em redes sociais. O cientista de dados entra em cena para **transformar esses dados em informações úteis**, que ajudam empresas a tomar decisões melhores e mais rápidas.

---

## 🧩 Quais áreas compõem a Ciência de Dados?

A Ciência de Dados é o encontro de três mundos:

1. **Programação / Computação**
   - Manipulação de dados com Python, SQL, etc.
   - Automatização de processos
   - Criação de modelos e algoritmos

2. **Estatística e Matemática**
   - Análise descritiva
   - Probabilidade
   - Testes estatísticos
   - Modelos matemáticos (regressões, distribuições, etc.)

3. **Conhecimento de Negócio**
   - Entendimento do problema que se quer resolver
   - Interpretação dos resultados
   - Comunicação com áreas não técnicas

---

## 🤔 Diferença entre Data Science, Data Analytics e Machine Learning

| Termo               | O que faz?                                                | Foco principal                     |
|---------------------|------------------------------------------------------------|------------------------------------|
| **Data Analytics**   | Analisa dados históricos para gerar relatórios e insights | Diagnóstico e análise              |
| **Data Science**     | Vai além da análise e constrói modelos preditivos         | Previsão e tomada de decisão       |
| **Machine Learning** | Subárea da ciência de dados focada em modelos que aprendem | Automatização e predição com dados |

➡️ **Resumo**:  
**Todo Machine Learning faz parte da Ciência de Dados**, mas **nem toda Ciência de Dados usa Machine Learning**.  
E **Data Analytics** está mais focado em análise exploratória e geração de relatórios, com menos programação e modelagem.

## 🛠️ Etapas de um Projeto de Ciência de Dados

Um projeto típico segue essas etapas:

1. **Entendimento do problema**
   - Qual a pergunta que precisamos responder?
   - Qual o objetivo de negócio?

2. **Coleta de dados**
   - Bancos de dados, APIs, planilhas, web scraping, sensores etc.

3. **Limpeza dos dados (Data Cleaning)**
   - Remoção de dados faltantes, duplicados ou inconsistentes

4. **Análise Exploratória (EDA)**
   - Visualizações, estatísticas descritivas, correlações

5. **Engenharia de atributos (Feature Engineering)**
   - Criação de novas variáveis relevantes

6. **Modelagem (Machine Learning)**
   - Escolha e treino de algoritmos

7. **Avaliação**
   - Métricas como acurácia, precisão, recall, F1-score etc.

8. **Deploy e Monitoramento**
   - Colocar o modelo em produção
   - Acompanhar desempenho e atualizações

## 👥 Perfis de Profissionais em Dados

Existem diferentes **perfis profissionais** dentro da área:

| Cargo                     | Principal foco                                 |
|---------------------------|------------------------------------------------|
| **Data Scientist**        | Modelagem estatística, machine learning        |
| **Data Analyst**          | Relatórios, visualizações e análise exploratória |
| **Data Engineer**         | Infraestrutura, pipelines e banco de dados     |
| **Machine Learning Engineer** | Deploy de modelos, produção, performance     |
| **BI Analyst**            | Dashboards e análise de indicadores            |

## 📌 Considerações Finais

- A Ciência de Dados é uma **área prática**: você aprende melhor praticando!
- Não é necessário saber tudo de início. Comece com o básico e evolua com consistência.
- Ferramentas como **Python**, **Pandas**, **Scikit-Learn**, e **Jupyter Notebooks** serão suas grandes aliadas.
